# 🍗 Управление доставкой продуктов — База данных

**Практическая работа по информатике**  
*Тема: «Проектирование, нормализация и работа с реляционными базами данных»*

Проект представляет собой систему управления **службой доставки продуктов питания**, разработанную на SQLite. База данных содержит информацию о клиентах, товарах и заказах на доставку по городу Воронежу.

---

## 📦 Технологии

- Язык: Python 3
- БД: SQLite
- Инструменты: SQLite Online, GitHub

---

## 🧱 Структура базы данных

### Таблица `users` (Покупатели)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Уникальный идентификатор читателя (Primary Key) |
| `name` | TEXT | Имя клиента |
| `age` | INTEGER | Возраст |

---

### Таблица `products` (Товары)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Уникальный идентификатор книги (Primary Key) |
| `nazvanie` | TEXT | Название продукта |
| `proizvoditel` | TEXT | Производитель |
| `country` | TEXT | Страна производства |
| `goden` | TEXT | Срок годности |

---

### Таблица `delivery` (Доставка)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | ID записи выдачи (Primary Key) |
| `user_id` | INTEGER | ID покупателя (Foreign Key к `users`) |
| `product_id` | INTEGER | ID товара (Foreign Key к `products`) |
| `address` | TEXT | Адрес получателя |
| `date` | TEXT | Плановая дата доставки |
| `status` | TEXT | Статус заказа (`Доставлен`, `В пути`, `Ожидает`) |
